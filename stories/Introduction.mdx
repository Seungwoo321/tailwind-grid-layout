# Tailwind Grid Layout

React Grid Layout의 Tailwind CSS 기반 대안입니다.

## Features

- **Drag & Drop** - 아이템을 드래그하여 위치 변경
- **Resize** - 8방향 핸들로 크기 조절
- **Responsive** - 브레이크포인트별 레이아웃
- **Collision** - 충돌 감지 및 자동 배치
- **Touch Support** - 모바일 터치 지원

## Installation

```bash
npm install tailwind-grid-layout
# or
pnpm add tailwind-grid-layout
```

## Quick Start

```tsx
import { GridContainer } from 'tailwind-grid-layout'
import type { GridItem } from 'tailwind-grid-layout'

const items: GridItem[] = [
  { id: '1', x: 0, y: 0, w: 4, h: 2 },
  { id: '2', x: 4, y: 0, w: 4, h: 2 },
]

function App() {
  const [layout, setLayout] = useState(items)

  return (
    <GridContainer
      items={layout}
      cols={12}
      rowHeight={60}
      onLayoutChange={setLayout}
    >
      {(item) => (
        <div className="bg-blue-500 text-white p-4 rounded">
          {item.id}
        </div>
      )}
    </GridContainer>
  )
}
```

## Components

| Component | Description |
|-----------|-------------|
| `GridContainer` | 기본 그리드 컨테이너 |
| `ResponsiveGridContainer` | 반응형 그리드 컨테이너 |
| `DroppableGridContainer` | 외부 드래그 앤 드롭 지원 |

## Utilities

| Utility | Description |
|---------|-------------|
| `generateLayouts` | 동일 레이아웃을 모든 브레이크포인트에 적용 |
| `generateResponsiveLayouts` | 컬럼 수에 맞게 레이아웃 자동 조정 |
| `WidthProvider` | 컨테이너 너비 자동 감지 HOC |
